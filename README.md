## 🏗️ Architecture Overview

This project follows a clean and scalable structure:

* **Frontend:** Lightning Web Components (LWC)
* **Backend:** Apex (Service Layer)
* **Data Layer:** Custom Objects
* **Platform:** Salesforce Experience Cloud (LWR/Aura)

### Key Patterns Used

* Service Layer Pattern
* Separation of Concerns (UI / Logic / Data)
* Reusable Components
* Apex Controllers with @AuraEnabled

---

## 📦 Data Model

### Custom Objects

* `Portfolio_Profile__c` → Personal information
* `Portfolio_Project__c` → Portfolio projects
* `Portfolio_Skill__c` → Technical skills
* `Portfolio_Experience__c` → Work experience
* `Portfolio_Contact__c` → Contact form messages

## 🚀 How to Run This Project

1. Clone this repository
2. Deploy metadata to your Salesforce org
3. Enable **Digital Experiences**
4. Create a site (LWR or Aura template)
5. Add LWC components in Experience Builder
6. Configure Guest User permissions
7. Publish the site
